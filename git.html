Cherry-pick

/* Cherry-pick: copy a specific commit to current branch */ git cherry-pick <hash> /* apply one commit */ git cherry-pick <hash1>..<hash2> /* apply a range */ git cherry-pick -n <hash> /* apply without committing */

Revert

✅ git revert — Safe Undo
A
→
B
→
C
→
C'

Adds a new commit C' that undoes C's changes. History is preserved — safe for team repos.

🔄 Revert of a Revert
A
→
C
→
C'
→
C''

Reverting C' creates C'' which re-applies C's changes. Critical: if you revert a feature branch merge and later re-merge the same branch, Git skips those commits — you must revert the revert first.

/* Revert: safe undo — adds a new "undo" commit */ git revert <hash> /* revert one commit */ git revert HEAD /* revert last commit */ git revert HEAD~3..HEAD /* revert a range of commits */ git revert <revert-hash> /* revert of revert → re-applies original */

Reset

⚠️ git reset — Rewrites History
A
→
B
→
C

--hard erases C from history. Using it on pushed commits forces divergence with remote — requires force push.

--soft: keep staged --mixed: keep unstaged --hard: discard all ⚠️
/* Reset: move HEAD (and branch pointer) — rewrites history */ git reset --soft HEAD~1 /* undo commit, keep staged */ git reset --mixed HEAD~1 /* undo commit + unstage (default) */ git reset --hard HEAD~1 /* undo commit + discard changes ⚠️ */ /* Recover a commit lost by reset (within ~30 days) */ git reflog /* find the lost commit hash */ git reset --hard <hash> /* restore to that commit */

/* fetch: download commits → origin/main only. Does NOT touch local main */ git fetch /* safe — no auto-merge */ git fetch origin /* fetch from origin explicitly */ git log origin/main ^main /* see what was fetched */ git merge origin/main /* merge manually after reviewing */ /* pull = fetch + merge (or rebase) in one step */ git pull /* fetch + merge */ git pull --rebase /* fetch + rebase (keeps history linear) */ /* push: send local commits to remote */ git push origin main /* push to origin main */ git push -u origin main /* set tracking branch and push */
