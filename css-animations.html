CSS Animations チートシート

@keyframes から transform まで、動くデモとコードで一緒に学ぼう。

1

@keyframes — アニメーションの定義

@keyframes bounceBall { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-80px); } } .ball { animation-name: bounceBall; animation-duration: 0.9s; animation-timing-function: ease-in-out; animation-iteration-count: infinite; /* shorthand: name duration timing count */ animation: bounceBall 0.9s ease-in-out infinite; }
2

animation プロパティ一覧

プロパティ 初期値 説明
animation-namenone使用する @keyframes の名前
animation-duration0s1 サイクルの時間（例: 1s, 300ms）
animation-timing-functionease速度カーブ（ease / linear / cubic-bezier など）
animation-delay0s開始までの待ち時間。負値で途中から開始可能
animation-iteration-count1繰り返し回数。infinite で無限ループ
animation-directionnormal再生方向 — normal / reverse / alternate / alternate-reverse
animation-fill-modenone終了後の状態 — none / forwards / backwards / both
animation-play-staterunning再生状態 — running / paused
/* shorthand: name duration timing delay count direction fill play-state */ .element { animation: fadeIn 0.5s ease-out 0.2s 1 normal forwards running; /* よく使うパターン */ animation: spin 2s linear infinite; animation: slideUp 0.4s ease-out forwards; animation: pulse 1.5s ease-in-out 0.5s infinite alternate; }
