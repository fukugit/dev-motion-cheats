# dev-motion-cheats

Animated cheat sheets for developers. Learn CSS and Git visually through live demos.

🔗 **https://fukugit.github.io/dev-motion-cheats/**

---

## Cheat Sheets

| Page | Contents |
|------|----------|
| [CSS Animations](https://fukugit.github.io/dev-motion-cheats/css-animations.html) | @keyframes, timing-function, transform, transition |
| [Git](https://fukugit.github.io/dev-motion-cheats/git.html) | add/commit 3-area diagram, branch/merge, push/pull |

## Features

- Understand commands and syntax through **live animated demos**
- Built with **HTML / CSS / Vanilla JS** — no external dependencies
- Clean, minimal design inspired by note.com

## Run Locally

```bash
git clone https://github.com/fukugit/dev-motion-cheats.git
cd dev-motion-cheats
open index.html   # macOS
```

## GitHub Pages Setup

Go to Settings → Pages → set Source to the `main` branch root.
