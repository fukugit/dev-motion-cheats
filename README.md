## 特徴

- コマンドや構文を **動くデモ** で視覚的に理解できる
- すべて **HTML / CSS / Vanilla JS** で完結（外部依存なし）
- note.com 風のシンプルで読みやすいデザイン

## ローカルで開く

```bash
git clone https://github.com/fukugit/dev-motion-cheats.git
cd dev-motion-cheats
open index.html   # macOS
```

## GitHub Pages の設定

Settings → Pages → Source を `main` ブランチのルートに設定するだけで公開できます。
